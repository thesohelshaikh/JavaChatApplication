
# JavaChatApplication
This is a simple chat application which utilizes Java Sockets.


SOFTWARE REQUIREMENTS
OS: Windows 7+, Ubuntu 14.0+ or MacOS.
IDE: Eclipse
Languages: Java
Database: MySql

HARDWARE REQUIREMENTS
Processor: Intel Core2Duo or higher.
Ram: 2GB or higher.
Disk Space: 50 GB or higher.

TECHNOLOGY USED
Eclipse IDE
IntelliJ IDEA IDE
MySQL Database
